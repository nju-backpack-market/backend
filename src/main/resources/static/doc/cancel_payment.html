DELETE /orders/{id}?name={name}&phone={phone}

Cancel a payment.

Parameter

  • id : order id
  • name: buyer name
  • phone: phone number

Note

  • name和phone用来验证是否是本人。因为只有本人才能根据姓名和电话号码搜索到订单而拿到订单编号。不验证的话随便填个id就可以取消订单，不安全。
  • 务必在取消订单时二次确认，并提示用户“您正在取消订单。取消后请千万不要在之前已打开了的付款页面上进行支付。否则，由此带来的不必要麻烦将由您自行承担”。如果他真的这么干了，如果是支付宝的话，三个小时内无法再通过对这个订单进行支付，不过款项应该会退回去的。
