PATCH /orders/{id}?status={status}

update order's status

Format

  • path variable: id
  • content type: application/x-www-form-urlencoded
  • parameter: status - the status which the order will change to

Example

PATCH /orders/1?status=STOCK_OUT

NOTE

当然Content-Type设为application/x-www-form-urlencoded然后参数放在Body里也ok，像这样。之后的文档就不重复说明了。

PATCH /orders/1/status HTTP/1.1
Content-Type: application/x-www-form-urlencoded

status=STOCK_OUT
