POST /products

Create products

Format:

 
​x
[
  {
    "name": [name],
    "price": [price],
    "description": [description]
  }...,
]

Example

 
x
[
  {
    "name": "MockProduct",
    "price": 10000,
    "description": "MOCK"
  },
  {
    "name": "MockProduct",
    "price": 10005,
  }
]

Note:

  • 不需要指定pid，指定了也会被忽略但不会返回错误
  • name和price是必选的，name不可为全空白，price不可为负数。
  • description是可选的，不填写则为空
  • 返回成功插入的实体的完整信息。不合法的对象会被忽略，但不会返回错误。
  • on_sale如果不填则为false，即下架状态
  • 权限接口。
