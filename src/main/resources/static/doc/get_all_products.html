GET /products

  • GET /products/{id} 获得单个商品

    返回指定的商品（如果存在的话），不管该商品是否处于上架状态

    不存在则404

  • GET /products?all={all} 获取所有商品

    参数: all 可选，默认为false，指示是否包括处于下架状态的商品。如果没有带有效的Token，那么all的值不会生效，始终为false。即只有管理员可以批量列出所有的商品

    不会404，没有则空列表

  • GET /products/{name}?all={all} 按名字搜索商品

    参数：all 同上

    返回值同上

  • DELETE /products/{id} 下架商品

    id可以是用逗号分隔的多个值
