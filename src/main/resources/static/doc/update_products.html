PUT /products

Update products

Format:

 
​x
[
  {
    "pid": [pid]
    "name": [name],
    "price": [price],
    "description": [description]
  }...,
]

Example

 
x
[
  {
    "pid": 1
    "name": "MockProduct",
    "price": 10000,
    "description": "MOCK"
  },
  {
    "pid": 2
    "price": 10005
  }
]

Note:

  • pid 必选。
  • 其他均为可选。
  • 只更新指定的字段
  • 返回成功更新的实体的完整信息。不合法的对象会被忽略，但不会返回错误。
  • 权限接口。
