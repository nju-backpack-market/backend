POST /orders

Create a new order.

Format:

 
{
  "bill": {
    "origin_total_price": [origin_total_price],
    "shopping_list": [
      {
        "pid": [pid],
        "count": [count],
        "origin_unit_price": [origin_unit_price],
        "origin_subtotal_price": [origin_subtotal_price]
      },...
    ]
  },
  "delivery_info": {
    "name": [name],
    "phone_number": [phone_number],
    "email": [email],
    "address": [address]
  }
}

Example:

 
xxxxxxxxxx
{
  "bill": {
    "origin_subtotal_price": 2533.28,
    "shopping_list": [
      {
        "pid": 1,
        "count": 2,
        "origin_unit_price": 1145.14,
        "origin_subtotal_price": 2290.28
      },
      {
        "pid": 2,
        "count": 3,
        "origin_unit_price": 81.0,
        "origin_subtotal_price": 243.0
      }
    ]
  },
  "delivery_info": {
    "name": "zhangsan",
    "phone_number": "1234567890",
    "email": "foo@bar.com",
    "address": "foobar"
  }
}

Response Code:

  • 409 CONFLICT: 请求中的包含价格不是最新价格

Note:

  • 事实上，直接把查询价格(POST /bills)的结果拿来用就行。
  • 需要在订单中写明价格的原因是为了防止在点击立即购买到点击付款（即在填写物流信息时），后台修改了商品价格。
  • 这一步若返回成功，请记录订单id，并跳转到选择支付方式的页面
